    Frequently Asked Questions
    How do I tune the squelch level?

    Go to the Settings tab and adjust the squelch slider. Lower dBFS values (more negative) make the scanner more sensitive (captures weaker signals but more noise). Higher dBFS values (closer to 0) reduce noise but may miss weak transmissions.

    The gold marker shows the recommended level based on your recent signal data. Start there and adjust up if you see too many noise segments, or down if you're missing activity.

    Clicking "Apply" restarts only the airband subprocess — it takes about 2 seconds and does not restart the container.

    Why am I not seeing any signals?

    Check these in order:

    1. Antenna — Is the antenna connected and suited for 2m/70cm? A mag-mount on a metal surface works best.
    2. Frequencies — Are you monitoring active repeaters in your area? Check RepeaterBook for local listings.
    3. Squelch — Too high a squelch will filter out everything. Try raising it toward -30 dBFS if you’re missing signals.
    4. RTL-SDR — Check the Debug tab — if "Process Alive" is false, the USB dongle may need a reset.
    What does "scanner parking" mean?

    RTLSDR-Airband monitors all configured frequencies simultaneously using FFT (no sequential scanning). Unlike traditional scanners, it never "parks" on one frequency — every channel is demodulated in parallel.

    If you see activity on only one frequency, it's because that's the only active repeater, not a scanning issue.

    How do I add or change frequencies?

    Edit the NODUS_EDGE_FM_CORE_FREQUENCIES field in the Settings tab (or directly in /app/.env). The value is a JSON array of frequencies in Hz, e.g.:

    [146940000, 147120000, 145230000]

    After saving, click "Restart Container" to apply the changes. The scanner will restart with the new frequency list.

    Can I run NodusEdge without internet (standalone mode)?

    Yes. NodusEdge works fully standalone. Without a NodusNet connection, segments are stored locally and displayed on the dashboard. You won't get repeater enrichment from the cloud, but the bundled repeater database covers most US repeaters.

    To connect later, set NODUSNET_SERVER in Settings and restart.

    Where are audio files stored?

    Audio files are stored in /data/captures/ inside the container. WAV files are kept for playback from the Live Feed tab. MP3 source files from RTLSDR-Airband are deleted after conversion unless NODUS_EDGE_FM_AIRBAND_KEEP_MP3=true is set.

    The capture directory is volume-mounted from the host, so files persist across container restarts.

    Standalone mode — Remote support requires a NodusNet connection. Local scanning and transcription work normally.
